# NHANES Health Data Analysis

This project uses Python and SQL to analyze NHANES data and explore functional movement difficulty among U.S. adults aged 20–65 through demographic and lifestyle indicators.

**Tools:** Python, SQL, SQLite, Jupyter Notebook

**Status:** Still work in progress

---

## Research Question

Which demographic and health factors are associated with 
self-reported functional movement difficulty in working-age 
U.S. adults (20–65)?

---

## Data Source

CDC National Health and Nutrition Examination Survey (NHANES)  
https://wwwn.cdc.gov/nchs/nhanes/

Downloaded four datasets from the 2017-2018 cycle: DEMO_J, BMX_J, 
PAQ_J, PFQ_J. Chose 2017-2018 because it's the last complete cycle 
before COVID disrupted data collection.
